# OrçaOS - Identidade Visual Dark Profissional

## Direção

A direção visual aprovada para o OrçaOS é dark profissional: técnico, moderno, confiável e direto para uso em campo.

O app deve parecer um sistema operacional do profissional técnico, não um ERP antigo de escritório e nem uma lista solta de calculadoras.

## Paleta Base

- Grafite: `#080f12`
- Cinza escuro: `#1a1f24`
- Teal: `#00b8c2`
- Verde: `#22c55e`
- Âmbar: `#f5a623`

Uso recomendado:

- Grafite e cinza escuro: fundos e superfícies.
- Teal: identidade, tecnologia, foco, navegação ativa, destaques institucionais.
- Verde: ações primárias, sucesso, aprovação e avanço de fluxo.
- Âmbar: atenção, prioridade, aviso e status intermediário.

## Tipografia

Inter deve continuar como referência tipográfica: legível, moderna e adequada para uso técnico no celular.

Pesos:

- Regular para texto comum.
- Medium/SemiBold para labels e descrições importantes.
- Bold para títulos, métricas e valores.

## Interface

Manter:

- tema escuro;
- superfícies com bordas discretas;
- cards técnicos limpos;
- botões grandes para toque;
- contraste alto;
- status claros;
- documentos e PDFs em branco profissional.

Evitar:

- fundos brancos fora de documento;
- blocos azuis soltos;
- excesso de gradiente;
- dashboard pesado;
- ícones decorativos sem função;
- textos longos em cards pequenos.

## Aplicação No Produto

O Início deve comunicar gestão e resumo do dia.

O fluxo principal deve permanecer claro:

Cliente → OS → Campo → Cálculo → Orçamento → Relatório

Os módulos devem explicar valor prático:

- Cálculos ajudam a decidir.
- Campo ajuda a levantar serviço e material.
- Orçamentos ajudam a vender com clareza.
- Relatórios ajudam a justificar tecnicamente.
- Catálogo ajuda a padronizar preço, material e serviço.

## Critério Visual

Ao abrir no celular, o OrçaOS deve transmitir:

- profissionalismo;
- confiança;
- organização;
- simplicidade;
- ferramenta pronta para atendimento real.
