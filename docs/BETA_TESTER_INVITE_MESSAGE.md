# OrçaOS - Mensagem para convidar testadores

Use como base para WhatsApp, e-mail ou lista de testadores da Play Console.

## Mensagem curta

```text
Oi! Estou abrindo o beta fechado do OrçaOS, um app para atendimento técnico em campo.

A ideia é testar o fluxo:
Cliente -> Atendimento -> Levantamento -> Cálculos -> Orçamento -> Aprovação -> OS.

Quero saber se a Home ficou clara, se o orçamento é fácil de montar e se o app parece útil em um atendimento real.

Se puder testar, me envie:
1. em que ponto ficou confuso;
2. se algum botão não funcionou;
3. se alguma tela ficou cortada ou travou;
4. se você usaria isso em campo;
5. quais cálculos ou campos fazem falta.

Obrigado por ajudar a validar antes de publicar para mais pessoas.
```

## Mensagem com instruções

```text
Oi! Você está recebendo a versão beta fechada do OrçaOS.

O que testar primeiro:
- abrir o app e conferir se a tela inicial mostra claramente o que fazer;
- criar um atendimento guiado;
- registrar levantamento com ambiente, serviço, material, medição e observação;
- usar um cálculo;
- montar orçamento;
- marcar como enviado e aprovado;
- conferir se "Converter em OS" aparece só depois da aprovação;
- fechar e abrir de novo para ver se os dados continuam salvos.

Importante:
- o app ainda está em beta;
- os dados ficam localmente no aparelho;
- financeiro é apenas gerencial, não fiscal;
- cálculos são apoio técnico e precisam de validação profissional.

Depois do teste, me mande uma nota de 0 a 10 para:
- clareza;
- utilidade em campo;
- aparência profissional;
- chance de usar em atendimento real.
```
