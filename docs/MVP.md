# MVP do OrçaOS

O MVP do OrçaOS deve provar que o produto é útil para o eletricista no dia a dia.

## Meta da primeira versão

Entregar uma aplicação simples, bonita e funcional com calculadoras elétricas confiáveis, catálogo de ferramentas, base de orçamento e estrutura preparada para evolução.

## Incluído no MVP 1

- Tela inicial.
- Catálogo de calculadoras.
- Corrente por potência.
- Potência por corrente.
- Consumo em kWh.
- Queda de tensão simplificada.
- Sugestão inicial de disjuntor.
- Sugestão inicial de cabo.
- Base de orçamento.
- Documentação de arquitetura.

## Não entra no MVP 1

- Autenticação.
- Sincronização em nuvem.
- Sistema de pagamento.
- PDF final.
- Relatórios com fotos.
- Cadastro persistente.
- Publicação nas lojas.

Esses recursos entram depois que a base técnica estiver validada.

## Critério de pronto

O MVP 1 estará pronto quando o app rodar localmente, os cálculos principais tiverem testes, os resultados forem validados manualmente, a interface funcionar bem no celular e as funções de cálculo estiverem separadas da tela.

## Primeira sequência de implementação

1. Rodar o app localmente.
2. Validar tela inicial.
3. Criar telas reais para cada calculadora.
4. Criar testes dos cálculos.
5. Ajustar visual mobile.
6. Criar orçamento simples.
7. Gerar PDF básico.
8. Preparar versão PWA ou Android.

## Regra de qualidade

É melhor ter poucas ferramentas corretas do que muitas ferramentas duvidosas.
