# Roadmap do Aferix

Este roadmap organiza a evolução do Aferix como ERP financeiro Local-First para prestadores de serviço e profissionais autônomos.

## Objetivo principal

Criar um aplicativo profissional para orçamento, lucro real, controle financeiro, relatórios, ordens de serviço e gestão comercial simples.

## Meta de produto

A primeira versão pública deve ser simples, bonita e útil. Ela não precisa ter tudo, mas precisa fazer corretamente o que promete.

Prioridade absoluta:

1. confiabilidade dos cálculos;
2. facilidade de uso;
3. organização do código;
4. visual profissional;
5. evolução gradual sem refazer tudo.
