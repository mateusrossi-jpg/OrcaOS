# Roadmap do OrçaOS

Este roadmap organiza a evolução do OrçaOS desde o MVP inicial até uma plataforma modular de gestão profissional.

## Objetivo principal

Criar um aplicativo profissional para eletricistas e técnicos, começando por ferramentas úteis de campo e evoluindo para orçamento, relatórios, ordens de serviço e gestão.

## Meta de produto

A primeira versão pública deve ser simples, bonita e útil. Ela não precisa ter tudo, mas precisa fazer corretamente o que promete.

Prioridade absoluta:

1. confiabilidade dos cálculos;
2. facilidade de uso;
3. organização do código;
4. visual profissional;
5. evolução gradual sem refazer tudo.
