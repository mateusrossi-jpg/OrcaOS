# Aferix - Formulário de feedback do beta

Use este roteiro com cada testador depois que ele instalar e usar a versão `0.1.0-rc.1`.

## Identificação

- Nome:
- Profissão:
- Cidade/estado:
- Modelo do celular:
- Versão do Android:
- Já usa algum app para orçamento/OS?

## Primeira impressão

1. Ao abrir o app, você entendeu para que ele serve?
2. A tela inicial deixou claro o que fazer primeiro?
3. O app parece profissional o suficiente para uso em atendimento real?
4. Alguma tela pareceu confusa, poluída ou pesada?

## Fluxo de atendimento

1. Você conseguiu criar um atendimento?
2. Você entendeu a diferença entre atendimento, orçamento e OS?
3. O levantamento substitui bem papel/caderno para uma visita técnica?
4. Faltou algum campo importante na visita?
5. O botão `Converter em OS` apareceu no momento certo, só depois da aprovação?

## Cálculos

1. Os cálculos essenciais foram fáceis de encontrar?
2. Algum cálculo ficou sem explicação suficiente?
3. As unidades estavam claras?
4. Você confiaria em usar os resultados como apoio, conferindo com sua experiência/norma?
5. Que cálculo faz mais falta para sua rotina?

## Orçamento

1. O orçamento manual rápido foi fácil de usar?
2. O orçamento guiado ficou claro?
3. O texto copiado/compartilhado ficou bom para enviar ao cliente?
4. Faltou algum campo comercial, como prazo, garantia, condição ou validade?
5. Você usaria esse orçamento em um atendimento real?

## Materiais e catálogo

1. O cadastro de material/serviço ficou fácil?
2. A lista de compra do cliente faz sentido?
3. Que materiais ou serviços você cadastraria primeiro?
4. A ideia de referência de compra online ajudaria ou atrapalharia?

## Financeiro e relatório

1. O financeiro simples ajuda a entender lucro real?
2. O relatório técnico parece útil para o cliente?
3. Faltou alguma informação importante no relatório?

## Problemas

1. O app travou, fechou sozinho ou ficou em tela branca?
2. Algum dado sumiu depois de fechar e abrir de novo?
3. Algum botão não funcionou?
4. Algum texto ficou cortado ou difícil de ler?
5. Em qual tela o problema aconteceu?
