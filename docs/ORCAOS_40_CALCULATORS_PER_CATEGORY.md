# Aferix — Varredura de Cálculos por Categoria

Objetivo: mapear pelo menos 40 cálculos por categoria principal para orientar a evolução do Aferix.

## Estratégia comercial

Cada categoria principal deve ter:

- 7 cálculos gratuitos;
- 20 a 40 cálculos totais na primeira expansão;
- cálculos avançados bloqueados no Pro;
- módulos muito específicos podendo virar pacotes próprios no futuro.

A versão gratuita deve ser útil de verdade, mas limitada o suficiente para gerar interesse no Pro.

# Conclusão

É viável trabalhar com pelo menos 40 cálculos por categoria principal, desde que a implementação seja feita em fases.

A recomendação é:

1. implementar 7 gratuitos por categoria principal;
2. completar 20 a 30 em categorias prioritárias;
3. expandir para 40+ conforme adesão;
4. transformar módulos de nicho em pacotes Pro específicos.
